<h1 align="center"> Jogos em Python </h1>

Esse repositório consta com dois jogos desenvolvidos em Python, sendo eles Forca e Adivinhação.

## :mag: Infraestrutura do Projeto

O arquivo "jogos.py" é o responsável por perguntar ao usuário qual jogo deseja jogar, chamando assim a função definida do jogo escolhido.

### :loop: Jogo da Forca

Corresponde ao arquivo "forca.py" , o próprio nome explica o arquivo, consiste em um jogo da forca em que o usuário deve adivinhar a palavra setada.

###  :loop: Adivinhação 

Consiste em um jogo de adivinhe o número com níveis diferentes de dificuldade variando o número de tentativa
